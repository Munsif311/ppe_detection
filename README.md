# ppe_detection
## This project implements a Personal Protective Equipment (PPE) Detection System using the YOLOv8 object detection model to automatically identify safety equipment in workplace environments. The system is designed to enhance occupational safety by detecting PPE items in real time, helping organizations monitor compliance with safety regulations and reduce workplace hazards.

The model was trained on a labeled PPE dataset containing various safety equipment classes and achieved strong detection performance with a Precision of 81.1%, Recall of 83.6%, and mAP@50 of 85.5%. The solution is capable of detecting PPE items in images and video streams, making it suitable for deployment in construction sites, factories, warehouses, and other industrial settings.

Built with Python, PyTorch, OpenCV, and Ultralytics YOLOv8, this project demonstrates the practical application of deep learning and computer vision for automated safety monitoring. The system can be further extended with real-time CCTV integration, violation alert generation, and edge-device deployment for smart workplace surveillance.

Key Highlights:

High detection accuracy (mAP@50: 85.5%)
Supports industrial safety compliance monitoring

Suitable for images, videos, and live camera feeds
Scalable for smart surveillance and workplace safety systems

This project showcases the effectiveness of state-of-the-art object detection techniques in improving workplace safety through automated PPE compliance monitoring.
